# Документация проекта FORMANIMA

Оглавление набора технической и проектной документации дипломного проекта **FORMANIMA** — веб-приложения для трекинга привычек и целей с геймификацией, модулями финансов и подсчёта калорий.

## Состав документации

| № | Документ | Описание |
|---|----------|----------|
| 00 | [Индекс документации](00-documentation-index.md) | Этот файл — оглавление набора |
| 01 | [Техническое задание](01-technical-specification.md) | Цель и назначение, целевая аудитория, 26 функциональных и 20 нефункциональных требований, ограничения, стек технологий |
| 02 | [Архитектура](02-architecture.md) | Общая архитектура, серверная и клиентская части, схема БД, JWT-аутентификация, система геймификации, схемы взаимодействия |
| 03 | [Тестирование](03-testing.md) | Стратегия тестирования, модульные тесты, функциональное тестирование API, тестирование безопасности, выявленные дефекты |
| 04 | [Целевая аудитория](04-target-audience.md) | Сегменты ЦА, анализ аналогов, методология и результаты опроса (N=15), выводы |
| 05 | [Экономическая эффективность](05-economic-efficiency.md) | Трудоёмкость и стоимость разработки, экономический эффект, срок окупаемости, индекс доходности |
| 06 | [Руководство пользователя](06-user-guide.md) | Пошаговые сценарии работы с приложением со скриншотами |
| 07 | [Установка и развёртывание](07-deployment.md) | Требования, запуск БД/сервера/клиента, переменные окружения, прод-сборка |
| 08 | [Схема базы данных (ERD)](08-database-schema.md) | ER-диаграмма (Mermaid) и справочник по всем моделям и ограничениям |
| 09 | [Справочник REST API](09-api-reference.md) | Все эндпоинты по модулям, авторизация, экспорт OpenAPI |

Скриншоты интерфейса находятся в каталоге [screenshots/](screenshots/).
