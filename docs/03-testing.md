# Результаты тестирования

## 1. Стратегия тестирования

Тестирование проекта FORMANIMA проводилось на трёх уровнях в соответствии с заданием на практику:

| Уровень | Инструмент | Объект тестирования |
|---------|-----------|---------------------|
| Модульное (unit) | Jest + NestJS Testing | Сервисный слой сервера |
| Функциональное | Ручное тестирование через Swagger UI | REST API эндпоинты |
| Безопасности | Анализ кода + тестирование входных данных | Auth, валидация, RBAC |

## 4. Тестирование безопасности

| № | Проверка | Механизм | Результат |
|---|----------|---------|-----------|
| SEC-01 | Защита маршрутов без JWT | `JwtAuthGuard` (глобальный) | 401 на любой защищённый эндпоинт |
| SEC-02 | Защита admin-эндпоинтов | `RolesGuard` + `@Roles('admin')` | 403 для роли USER |
| SEC-03 | Хеширование паролей | bcrypt, cost factor 10 | Пароли не хранятся в открытом виде |
| SEC-04 | Валидация входных данных | `ValidationPipe` (whitelist, transform) | Неизвестные поля отброшены, типы приведены |
| SEC-05 | CORS | Разрешён только `http://localhost:5173` | Запросы с других origin отклонены |
| SEC-06 | Владелец ресурса | Проверка `userId` в каждом сервисном методе | ForbiddenException при несовпадении |
| SEC-07 | Refresh token | Хранится как bcrypt-хэш в БД | Нельзя использовать после логаута |
| SEC-08 | SQL-инъекция | Prisma ORM, параметризованные запросы | Прямая конкатенация SQL отсутствует |

## 5. Выявленные и исправленные дефекты

| ID | Описание дефекта | Серьёзность | Статус |
|----|-----------------|-------------|--------|
| BUG-001 | `GoalProgress` не имеет `@@unique` с nullable `milestoneId` — PostgreSQL считает `NULL != NULL`, что позволяло создавать дубликаты | Значительный | ✅ Исправлен: использован `findFirst` вместо `findUnique` |
| BUG-002 | Моки в `gamification.service.spec.ts` ссылались на устаревшие имена моделей (`completion`, `habit`) вместо актуальных (`goalProgress`, `goal`) | Значительный | ✅ Исправлен: моки обновлены |
| BUG-003 | `bcrypt.hash(password, 10)` в тестах вызывал таймаут 5000 ms при реальных вычислениях | Незначительный | ✅ Исправлен: bcrypt замокан в тестах |
